SDMetrics sample output files

This folder contains a sample XMI file and output files generated with the SDMetrics export feature (which is disabled in the demo version). Use these files to test the suitability of SDMetrics' output files.

The files in this archive are:

File/FolderDescription
sample.xmi The XMI source file from which the output was calculated using the default metamodel, metrics, and XMI import. Element and rule filters were disabled.
Metric data, single file
sample.txtTab-separated text
sample.csvComma-separated vectors
sample.htmlHTML file
sample.odsOpenDocument Spreadsheet (OpenOffice 2.x etc)
sample.xmlXML for Microsoft Excel 2002
sample.scxOpenoffice.org 1.0 Spreadsheet
Metric data, separate files
TXT-separateFiles/Tab-separated text
CSV-separateFiles/Comma-separated vectors
HTML-separateFiles/HTML file (open file sample_index.html).
SXC-separateFiles/Openoffice.org 1.0 Spreadsheet
XML-separateFiles/XML for Microsoft Excel 2002
Miscellaneous data output features
DumpModel/Output of "Model Dump", separate .txt files
descstats.htmlOutput of descriptive statistics, single .html file
RelMatrices/Output of relation matrices, separate .txt files
Rules/Output of the design rule checker, separate .html files (open file sample_index.html)
Graph export features
Kiviat/All Kiviat diagrams as SVG, with "html page for all" option. Open file sample_index.html for results with "use frames" option, open file sample.html for result without frames (large page, takes a while to load). To view the graphs in a web browser, the browser must support the SVG format (usually requires installation of a browser plugin).
Histo/All package histograms as PNG, with "html page per type" option. Open file sample_index.html for results with "use frames" option, open file sample.html for result without frames.
CumDist/Cumulative distribution for interfaces as JPGs, with "html per diagram" option.

Model dump, descriptive statistics, rules, and relation matrices can also be exported in any of the other formats. Export of data from design comparisons produces files of the same structure as the regular metric data and descriptive statistics output.

If you need to see a specific combination of output features not covered above, send an e-mail to info@sdmetrics.com.
